Data Format

Each item in collection.json looks like this:

{
  "id": "sotm-us-2019",
  "name": "State of the Map US 2019",
  "category": "event",          // event | concept | company | collection
  "year": 2019,
  "location": "Minneapolis, MN",
  "description": "The annual US OpenStreetMap conference…",
  "tags": ["openstreetmap", "sotmus"],
  "photos": [
    {
      "file": "sotm-us-2019-front.jpg",
      "caption": "Front of the shirt",
      "credit": "Photo by @iandees"
    }
  ],
  "stories": [
    {
      "author": "Your Name",
      "text": "I wore this shirt to my first mapping party…"
    }
  ]
}
